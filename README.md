# My_SQL

This repository contains MySQL programs and scripts used for database creation, data manipulation, and querying. The files include SQL commands for creating databases, tables, inserting data, and performing various SQL operations, intended for learning, practice, and academic use.

## Features

- MySQL scripts for database and table creation
- Data insertion, update, and deletion queries
- SQL queries using joins, subqueries, and constraints
- Useful for practice, labs, and interview preparation

## Prerequisites

- MySQL Server
- MySQL Workbench or any MySQL-compatible client
- Basic knowledge of SQL

## Usage

1. **Clone the Repository** to your local machine:
2. 
   https://github.com/your-username/your-repo-name

3. **Open MySQL Workbench or any MySQL client**

4. **Open the required `.sql` file** from the repository

5. **Execute the queries** on your MySQL database

## How it Works

1. The repository contains MySQL script files (`.sql`).
2. Each file includes SQL commands executed sequentially.
3. Users can run scripts to create or modify databases and tables.
4. Queries can be executed individually or as a complete script.

## Example

Run a SQL file in MySQL:

SOURCE path/to/file.sql;
